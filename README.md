# TDD Console String Library (C++)

## Тема
Библиотека для работы со строками + консольное меню.

## Функции
- reverseString
- toUpperCase
- toLowerCase
- isPalindrome
- countVowels

## Структура
src/ - библиотека
tests/ - unit tests
main.cpp - консольное меню

## TDD подход
Сначала написаны тесты, затем реализация.
